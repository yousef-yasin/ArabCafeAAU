# Arab Cafe - نسخة الصور داخل الداتا بيس

## المهم في هذه النسخة
- صور الشعار والمنيو وصور الأصناف الأساسية **تتخزن داخل قاعدة البيانات**.
- الأدمن يقدر يضيف صنف جديد مع صورة، أو يعدل اسم الصنف وسعره ووصفه وصورته.
- عند النشر على دومين وسيرفر، الموقع يعرض الصور من قاعدة البيانات نفسها.
- المنيو كامل، الطلبات، والتقارير اليومية كلها موجودة.

## التشغيل
1. فك الضغط عن المشروع.
2. افتح Terminal داخل مجلد `cafeteria_db_ready`.
3. ثبّت المكتبات:

```bash
pip install -r requirements.txt
```

4. شغّل المشروع:

```bash
python app.py
```

5. افتح:
- موقع الطلاب: `http://127.0.0.1:5000/`
- لوحة الكافتيريا: `http://127.0.0.1:5000/admin/login`

## دخول الأدمن
- username: `admin`
- password: `admin123`

## إذا كنت مشغّل نسخة قديمة
احذف ملف `cafeteria.db` القديم أو شغّل هذه النسخة في مجلد جديد حتى تبدأ قاعدة البيانات الجديدة بشكل نظيف.

## الطباعة
- عند تأكيد الطلب تفتح صفحة الفاتورة وتعمل print تلقائياً.
- على جهاز الكافتيريا، اجعل الطابعة الحرارية هي الافتراضية.

## ملاحظات تقنية
- قاعدة البيانات: SQLite
- الصور تُخزن داخل الجداول كـ BLOB
- تم وضع ملفات seed داخل مجلد `seed` فقط من أجل تعبئة قاعدة البيانات أول تشغيل
- بعد أول تشغيل، عرض الصور في الموقع يكون من قاعدة البيانات عبر routes مثل `/media/item/...`
